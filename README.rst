A library that gets infos from APK.

Install and Test
^^^^^^^^^^^^^^^^

.. code-block::

   $ pip install apkutils

Or

.. code-block::

   $ pip install git+https://github.com/codeskyblue/apkutils

Usage
^^^^^

.. code-block::

   $ python -m apkutils -h
   usage: apkutils [-h] [-m] [-s] [-f] p

   positional arguments:
     p           path

   optional arguments:
     -h, --help  show this help message and exit
     -m          Show manifest
     -s          Show strings
     -f          Show files

.. code-block::
